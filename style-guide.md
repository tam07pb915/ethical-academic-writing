# 文体・ムーブ・時制ガイド

## 1. セクション別の時制ルール

| セクション | 基本時制 | 使い分け |
|-----------|---------|---------|
| Introduction | 現在形＋過去形 | 一般知見→現在形，特定の先行研究→過去形/現在完了 |
| Methods | 過去形 | 実施済みの手順すべて |
| Results | 過去形 | 分析結果の報告 |
| Discussion | 現在形＋過去形 | 解釈・含意→現在形，結果の再掲→過去形 |

### Reporting Verbsの時制

- 特定の著者＋年号つき → **過去形**（Smith (2020) found...）
- 複数研究をまとめた引用 → **現在完了形**（Several studies have shown...）
- 確立された一般的知見 → **現在形**（Research suggests...）

## 2. CARSモデル（Introduction）

**Move 1: 研究領域の確立**
- X has been widely studied...
- Recent research has shown that...
- There is growing interest in...

**Move 2: ニッチの確立（ギャップ）**
- However, few studies have examined...
- Little is known about...
- Previous research has not adequately addressed...

**Move 3: ニッチの占拠（目的）**
- The present study aims to...
- This paper investigates...
- The purpose of this study is to...

## 3. Abstractの5ムーブ

| ムーブ | 典型表現 | 分量目安 |
|-------|---------|---------|
| Background | X has become increasingly important... | 10-15% |
| Purpose | The present study aims to... | 1文 |
| Method | Data were collected from... | 20-25% |
| Results | The results showed that... | 30-35% |
| Conclusion | These findings suggest that... | 15-20% |

## 4. 文体ルール

### 主語を短くし動詞を早めに出す

❌ The results of the analysis of the data collected from the participants showed...
✅ The analysis results showed...

### ヘッジとブースター

| 強さ | ヘッジ（弱い主張） | ブースター（強い主張） |
|-----|-----------------|-------------------|
| 助動詞 | may, might, could | must, will |
| 動詞 | suggest, indicate, appear to | demonstrate, confirm, establish |
| 副詞 | possibly, perhaps | clearly, certainly |

### 受動態・能動態の使い分け

- 判断・決定を明示 → 能動態（We selected...）
- 手順の一般的記述 → 受動態（The data were collected...）
- 著者の主張 → 能動態（We argue that...）

### 名詞化（Nominalization）

動詞→名詞の変換で情報を圧縮：analyze→analysis, differ→difference, improve→improvement

### 日本語話者の冗長表現

| 冗長 | 簡潔 |
|-----|-----|
| in order to | to |
| due to the fact that | because |
| it is possible that | possibly / may |
| a large number of | many |
| at the present time | currently / now |

## 5. Reporting Verbsのスタンス分類

| スタンス | 動詞例 |
|---------|-------|
| 中立的報告 | reported, found, observed, noted |
| 積極的主張 | argued, claimed, maintained, asserted |
| 示唆・提案 | suggested, indicated, implied, proposed |
| 実証・証明 | demonstrated, confirmed, established, showed |
| 批判・疑問 | questioned, challenged, criticized, disputed |

## 6. Discussion の6ムーブ

| ムーブ | 典型表現 |
|-------|---------|
| 結果の要約 | The findings indicate that... |
| 結果の解釈 | A possible explanation is that... |
| 先行研究との比較 | This finding is consistent with... / In contrast to... |
| 含意の提示 | These results have implications for... |
| 限界の記述 | A limitation of this study is that... |
| 今後の研究 | Future research should examine... |

## 7. 接続表現のフォーマル度

| 機能 | フォーマル | インフォーマル（避ける） |
|-----|----------|---------------------|
| 追加 | In addition, Furthermore | Also, Plus |
| 対比 | However, Nevertheless | But, Though |
| 因果 | Therefore, Consequently | So |
| 例示 | For instance, For example | Like |

## 8. パラグラフの内部構造（TSSC）

1. **T**opic sentence — そのパラグラフの主張
2. **S**upporting sentences — 根拠・説明・データ
3. **S**ummary / transition — まとめと次への橋渡し
4. **C**ohesion — Given-New原則で文と文をつなぐ

## 9. 統計報告書式（APA第7版）

- t検定: *t*(df) = X.XX, *p* = .XXX, *d* = X.XX
- ANOVA: *F*(df1, df2) = X.XX, *p* = .XXX, η² = .XX
- 相関: *r*(df) = .XX, *p* = .XXX
- カイ二乗: χ²(df) = X.XX, *p* = .XXX

効果量は必ず報告する。
